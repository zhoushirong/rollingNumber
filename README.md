# rollingNumber

### 用途

创建一个能够滚动的数字

### Example

``` javascript

	var scrollNum1 = new ScrollNum();
	//初始化滚动数字组件
	scrollNum1.init({
		fBoxId: "numBox1",
		tagH: 20, // num的高度
		numLen: 9, //滚动数字的位数
		num: 123456789 // 初始数字
```
